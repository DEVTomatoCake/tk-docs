lang: en

# Create ticket

`ticket [<Reason>]`

# Add user to ticket

`add <User>`

# Remove user from ticket

`remove <User>`

# Close ticket

`close [Reason]`

# Delete ticket

`delete`

In order to delete a ticket you need the "Manage Server" permission.

# Reopen a closed ticket

`reopen`

# Save transcript

`transcript`

# Show tickets & transcripts

`tickets [<User>]`
